Closure, Lifetime, and HRT

01 Mar 2017, 00:00
rust / closure / HRT

Intro So here is the thing. Can we write something like let mut vector = vec![1, 2, 3]; let mut closure = |x| vector.get_mut(x); println!("{:?}", closure(2)); in Rust? The answer is, unfortunately, no. Compiling it as of rust 1.15.1 would yield some error message like: error[E0495]: cannot infer an appropriate lifetime for lifetime parameter in function call due to conflicting requirements --> src\main.rs:100:23 | 100 | let closure = |x| vector.
